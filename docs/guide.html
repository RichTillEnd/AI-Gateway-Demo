1. 帳號與登入

使用公司信箱註冊，完成 Email 驗證即可開始使用。

1

註冊帳號

前往登入頁面，點選「註冊」分頁。輸入使用者名稱、公司信箱（@ystravel.com.tw）及密碼。密碼需包含大小寫字母、數字與特殊符號，長度至少 8 碼。

2

Email 驗證

註冊後系統會發送驗證信至你的信箱，點擊信中連結完成驗證。驗證連結有效期為 24 小時，過期需重新註冊。

3

登入系統

註冊完畢後回到登入表單，輸入註冊好的用戶名稱及密碼即可登入

4

開始對話

登入後進入主畫面，即可在輸入框輸入問題開始與 AI 對話。

密碼每 90 天需更換一次。系統會在到期前提醒你，請於登入頁面下方有「重設密碼」選項進行重設密碼。

2. AI 對話

支援即時串流回覆、多模型切換、知識庫檢索與網路搜尋。

即時串流對話

訊息以即時串流方式逐字顯示，不需等待整段回覆完成。支援 Markdown 格式、程式碼區塊語法高亮。

多模型選擇

可在 OpenAI GPT 與 Google Gemini 之間自由切換。開啟「自動路由」後，系統會根據問題類型智慧選擇最適合的模型。

對話操作

  • 傳送訊息

    在底部輸入框輸入文字，按下 Enter 或點擊傳送按鈕。換行請使用 Shift + Enter。

  • 新對話

    點擊左上角的「新對話」按鈕或使用快捷鍵開啟新的對話窗口。每段對話會自動依據內容生成標題。

  • 對話管理

    側邊欄顯示所有歷史對話。你可以重新命名、加星號（收藏）、移入專案，或刪除不需要的對話。也支援跨對話的全文搜尋。

  • 網路搜尋

    AI 會即時搜尋網路取得最新資訊來回答問題，適用於需要時事或即時數據的場景。

使用「Prompt 範本」功能可快速套用預設的提示語模板，省去每次重新撰寫系統指令的時間。

3. 檔案與圖像

上傳文件讓 AI 閱讀分析，或使用 AI 生成圖像。

文件上傳

支援 PDF、Word、Excel、CSV、程式碼等格式。AI 會自動擷取文件內容並結合你的問題進行分析。單檔上限 50MB。

圖片分析

上傳圖片後，AI 可辨識並描述圖片內容（視覺分析）。支援 JPG、PNG 等常見格式，可用於圖表解讀、截圖分析等情境。

AI 圖像生成

使用自然語言描述你想要的圖像，AI 會為你生成。支援 OpenAI 與 Google Imagen 兩種引擎，可選擇尺寸與品質。

4. 進階功能

善用這些功能，讓 AI 更精準地為你服務。

RAG 知識庫

AI 回答時會自動搜尋公司內部文件作為參考依據，附上引用來源。讓回答更準確、更貼近公司實際情境。

人事 政策 技術 一般

AI 記憶

開啟記憶功能後，AI 會自動記住你的偏好與工作背景。例如記住你的部門、常用工具、溝通風格等，讓之後的對話更個人化。你可隨時查看或刪除已儲存的記憶。

網路搜尋

需要最新資訊時開啟此功能。AI 會透過搜尋引擎取得即時資料輔助回答，適合查詢新聞、天氣、匯率、法規更新等。

Prompt 範本

系統預設了多種提示語範本，包含翻譯、摘要、寫作等場景。選擇範本後自動填入提示語，你只需補上具體內容即可。

個人化指令

在設定中可自訂「個人指令」，例如：「請用繁體中文回答」、「保持簡潔」。此指令會在每次對話中自動帶入，不需重複輸入。

自動歷史摘要

當對話訊息超過一定長度時，系統會自動將較早的對話內容進行摘要壓縮，確保 AI 在有限的上下文窗口內仍能掌握完整脈絡。

5. 專案管理

將相關對話分組到專案中，設定專屬指令與知識庫。

  • 建立專案

    在側邊欄點擊「新增專案」，為一組相關的對話建立分類。例如為不同客戶案件、部門任務各建一個專案。

  • 專案指令

    為每個專案設定專屬的系統指令（System Prompt），該專案下的所有對話都會自動套用。例如為翻譯專案設定「請以正式商業用語翻譯」。

  • 專案知識庫

    可為專案上傳私有文件（TXT、PDF、Word、Markdown、Excel、CSV），AI 在該專案的對話中會優先參考這些文件。最大 20MB/檔。

  • 移動對話

    任何對話都可以隨時移入或移出專案。刪除專案時，對話不會被刪除，僅解除分組關係。

你也可以指定專案可存取的 RAG 知識庫分類，讓 AI 在專案對話中只參考相關類別的文件，避免資訊干擾。

6. 個人設定

調整介面外觀、AI 行為偏好與帳號安全。

AI 模型

選擇偏好的 AI 供應商（OpenAI / Gemini），或啟用自動路由讓系統智慧選擇。

外觀主題

支援淺色與深色模式切換。可選擇多種中文字體，調整文字大小至最舒適的閱讀體驗。

個人指令

設定你的工作類型與自訂指令，AI 會在每次回覆中自動參考這些偏好。

帳號安全

可隨時修改密碼。系統每 90 天要求更換密碼以確保帳號安全。

功能開關

AI 記憶、RAG知識庫、Webhook等功能，依照使用場景靈活調整。

記憶管理

查看 AI 記住的所有關於你的資訊，可逐條刪除或一次清除，確保隱私掌控在你手中。

7. 管理後台

僅限管理員。提供用戶管理、用量監控與系統設定功能。

管理後台僅限擁有管理員權限的帳號存取。

用戶列表 — 檢視所有註冊用戶，包含帳號、信箱、部門、啟用狀態、管理員權限。可逐一編輯或刪除。

批次匯入 — 下載 Excel 範本，填入用戶資料後上傳即可批次建立帳號。也支援 CSV 格式。未填密碼欄位的用戶會自動生成臨時密碼，首次登入時須更換。

強制改密碼 — 可強制指定用戶在下次登入時更換密碼。

月度配額 — 為每位用戶設定每月 Token 上限和費用上限。達到上限後用戶將無法發送新訊息，直到下月重置。

速率限制 — 預設每分鐘 5 次、每小時 60 次、每日 100 次請求。管理員可重置特定用戶的計數器。

手動重置 — 可隨時重置用戶的當月使用量或速率限制計數器。

總覽儀表板 — 顯示全站用戶數、對話數、訊息數、30 天費用與 API 呼叫次數。

趨勢圖表 — 7 天每日訊息量折線圖、模型使用分佈圓餅圖。

使用日誌 — 逐筆 API 呼叫記錄，含用戶、供應商、模型、Token 數與費用。

部門成本分攤 — 按部門彙整費用，含每位用戶明細與供應商分佈。可匯出 Excel 報表。

模型效能 — 各模型的請求次數、Token 消耗量與費用統計。

文件管理 — 上傳文件至知識庫（PDF、Word、TXT、Markdown、Excel），也可直接貼上文字。每份文件可指定分類與標題。支援預覽、下載原始檔及刪除。

分類管理 — 建立與刪除知識庫分類。預設分類：人事、政策、技術、一般。含有文件的分類無法直接刪除。

存取控制 — 為每位用戶設定可存取的知識庫分類。未設定時預設可存取全部分類。

設定關鍵字觸發的自動回覆規則。當用戶訊息命中關鍵字時，系統直接回傳預設答案，不消耗任何 AI Token。

適合用於：常見問答（如公司 WiFi 密碼、報帳流程）、公告通知等高頻重複問題。支援正則表達式匹配，並追蹤每條規則的命中次數。

Prompt 範本管理 — 建立、編輯、排序與停用提示語範本，供全公司用戶使用。

API Key 管理 — 建立外部 API Key 供第三方系統串接，可設定速率限制與費用上限。

錯誤日誌 — 檢視系統錯誤記錄，含堆疊追蹤、端點、時間戳。可標記為已解決。

審計日誌 — 完整的操作審計軌跡，可依操作者、動作類型、資源類型與日期範圍篩選。

PII 偵測設定 — 配置個資偵測規則，自動遮蔽用戶輸入中的敏感資訊。

語意快取 — 查看快取命中率與統計資訊，可清除快取以釋放空間。

8. 常見問題

可能原因：

1. 速率限制 — 每分鐘最多 5 次、每小時 60 次、每日 100 次。

2. 配額用盡 — 當月 Token 或費用已達上限。可在設定中查看剩餘額度，或聯繫管理員重置。

3. 帳號停用 — 請確認信箱中的驗證信已點擊。

OpenAI GPT — 整體表現穩定，擅長程式碼生成、複雜推理與結構化輸出。

Google Gemini — 回應速度較快，費用較低，擅長多語言翻譯與長文摘要。且內建 Google 搜尋能力。

建議：開啟「自動路由」讓系統根據問題類型自動選擇最合適的模型。

1. 確認問題與知識庫中的文件內容語義相關，而非僅靠關鍵字匹配。

2. 檢查文件是否已歸入正確的分類，且你的帳號擁有該分類的存取權限。

3. 若使用專案知識庫，確認文件已上傳至對應的專案中。

4. 聯繫管理員確認知識庫文件是否已正確向量化。

AI 記憶僅記錄你在對話中明確表達的偏好與背景資訊（例如：部門、職稱、常用工具等），不會記錄對話的完整內容。

你可以隨時在設定中查看所有記憶內容、逐條刪除或完全停用記憶功能。系統也啟用了 PII 偵測，會自動遮蔽身分證號、電話等敏感資訊。

在主介面的設定面板中可以查看：

1. 配額狀態 — 當月已使用的 Token 數及百分比剩餘額度。

2. 速率狀態 — 當前各時段（分鐘/小時/日）的剩餘請求次數。

管理員可在後台為你調整配額或重置計數器。

在登入頁面點擊「忘記密碼」連結，輸入你的註冊信箱。系統會發送重設密碼的連結至信箱，連結有效期為 30 分鐘。

訊息回覆速度取決於當前供應商API負載工作量及問題的複雜程度和當下的網路流量，問題越複雜、回答越長，就需要越多時間思考。這是正常現象，而非系統故障。 若回覆訊息中有提示明確問題說明請聯繫 IT 部門。
